
HeightmapToRGBA Node
====================


HeightmapToRGBA converts a series of heightmaps into an RGBA splatmap.



![img](../../images/nodes/HeightmapToRGBA_settings.png)


# Category


Converter
# Inputs

|Name|Type|Description|
| :--- | :--- | :--- |
|A|VirtualArray|Alpha channel.|
|B|VirtualArray|Blue channel.|
|G|VirtualArray|Green channel.|
|R|VirtualArray|Red channel.|

# Outputs

|Name|Type|Description|
| :--- | :--- | :--- |
|RGBA|VirtualTexture|RGBA heightmap.|
